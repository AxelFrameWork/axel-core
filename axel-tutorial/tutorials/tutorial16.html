${tutorial_page_index} example - script call

The script action will make a javascript call and return the result.

How to run simple javascript contained in the script content
${instruction-counter}. Edit the index.html page and add the following content
                
                    

	var message = 'Hello World!!!';
	message;
Runs the javascript and returns the response 'Hello World!!!' which is displayed on screen.
${instruction-counter}. Refresh the browser and now you see the new content from the script response.
How to run javascript from a file
${instruction-counter}. Create a new folder called js. We use this as a standard location for javascript files.
${instruction-counter}. Create a new file called msg.js. This is the javascript file we will call from the script action.
${instruction-counter}. Edit the msg.js file and add the following content
                
                    
var message = 'Hello World!!! from msg.js';
message;
${instruction-counter}. Edit the index.html page and add the following content
Runs the javascript file msg.js and returns the response 'Hello World!!! from msg.js' which is displayed on screen.
${instruction-counter}. Refresh the browser and now you see the new content from the script response.
