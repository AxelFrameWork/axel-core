How to setup Axel with Springboot

These instructions show how to configure Axel to run with Springboot. The steps include

  • How to setup a gradle build script
  • Dependent libraries required to run Axel and Springboot
  • Building a runnable war file
  • Running Springboot and accessing Axel using localhost

    • The resultant war file will also run inside a docker container. Instructions for running a war inside a docker container are not included with this setup, there are many sites that provide these instructions.
